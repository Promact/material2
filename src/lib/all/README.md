#md2 (Angular2 Material)

## Installation
`npm install --save md2`

## API

Example:
 
  ```ts
//app-module.ts

import {Md2Module} from '@angular2-material/all';

@NgModule({
  imports: [
    Md2Module.forRoot(),
  ],
  declarations: [
    ...
  ]  
})
 ```
